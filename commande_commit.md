## git commit

**git commit** - перенести изменения всех файлов в репозиторий.

**Коммит** - точка фиксации изменений.

При создании коммита требуется указать сообщение.

*Параметр -m* позволяет написать сообщение из коммандной строки.

***Пример:*** 

`git commit -m "init"`
